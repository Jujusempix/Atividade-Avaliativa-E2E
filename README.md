# CRUD de Produtos

## Requisitos

- Node.js 20+
- npm

## Instalacao

Na raiz do projeto, instale as dependencias de cada pasta:

```bash
cd Backend
npm install

cd ../FrontEnd
npm install

cd ../cypress
npm install
```

## Executar para testar

1. Inicie o backend:

```bash
cd Backend
npm run dev
```

2. Em outro terminal, inicie o frontend:

```bash
cd FrontEnd
npm run dev
```

3. Em outro terminal, execute os testes E2E:

```bash
cd cypress
npm run dev
```
